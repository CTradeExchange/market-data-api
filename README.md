# Market-Data-API

# 开发者的第一个行情API

一个简单好用易上手,产品齐全，接入友好的行情API

## 功能特性：

- 免费开源

- 获取港股实时行情

- 获取港股实时一档盘口

- 获取港股K线数据

- 获取美股实时行情

- 获取美股实时一档盘口

- 获取美股K线数据

- 订阅港股延时行情

- 订阅美股延时行情
